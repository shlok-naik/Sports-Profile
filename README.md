# CombineAI - United Hacks V7

## How to Run (Optional Live Demo for Project)
### Version
To run this app, you must use python versions 3.9-3.12 (3.9.13 is the one that works most reliably for us).

### Enviroments
After installing this, you can create a venv to store the requirements `py -3.9 -m venv venv` (recommended) or do it on your global python save.

#### If you did the venv route, do these next to activate your venv:
Windows - `venv\Scripts\activate`

Mac/Linux - `source venv/bin/activate`

### Installation
You must run `pip install -r requirements.txt` to install required modules.

### Run
Then you can use `python app.py` to run the program on http://127.0.0.1:5001.
